# ProposalPilot — Prompt to Production Demo

A public attendee repo for the webinar **From Prompt to Production: Building SaaS with AI in Under 1 Hour**.

If this helps you build your first AI-powered SaaS slice, kasih ⭐ biar repo ini gampang ditemukan peserta lain.

## Webinar Angle

Developer tidak harus cuma nunggu project, invoice, atau client work untuk produktif. Di era AI, skill ngoding bisa dipakai untuk mulai membangun produk SaaS sendiri — kecil dulu, tapi nyata, usable, dan bisa divalidasi.

## Demo Journey

```text
Landing
→ Pricing
→ Register/Login demo state
→ Choose Plan
→ Mock Checkout
→ Subscription Active
→ Quota Available
→ Generate Proposal
→ Save Proposal
→ Quota Decreases
→ Track Proposal Status
```

## What Is Implemented

- Polished landing page
- Pricing cards: Starter, Pro, Agency
- Demo-auth state activated by mock checkout
- Clearly labeled mock checkout flow
- Subscription active state
- Monthly quota state
- Proposal brief form with seed/demo data
- Deterministic AI proposal generation fallback
- Proposal save/list
- Quota decrement after successful generation/save
- Proposal status update: Draft, Sent, Accepted, Rejected
- Playwright E2E proof for the main journey and pre-checkout guardrail

## Honest Demo Cuts

These are intentionally **not** implemented in the under-1-hour live build:

- real recurring billing
- payment webhooks
- invoices
- admin dashboard
- team workspace
- forgot password
- cancellation/proration
- durable database persistence

## Production Hardening Next Steps

- Replace demo-auth with real auth
- Add database-backed users, subscriptions, usage ledger, and proposals
- Replace mock checkout with a billing provider and webhook handling
- Add server-side quota and entitlement checks
- Add AI provider abstraction with retries, cost limits, and logging
- Add tests for auth, entitlement, quota decrement, and proposal ownership
